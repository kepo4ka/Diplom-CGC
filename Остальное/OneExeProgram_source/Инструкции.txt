Как скомпилировать сборку exe и зависих dll-ок в один файл exe

1) Сжать (по желанию) нужную библиотеку. 
	Для этого запихать её в папку scripts и запустить файл с расширением ".linq" с помощью программы LINQPad, получим файл с расширением "dll.deflected"
	
2) Добавить в ссылках проекта ссылку на исходную библиотеку, она может находится в любом месте.

3) Открыть свойства Проекта и в Ресурсах добавить файл из 1 пункта - "dll.deflected" 
4) Добавить в функции AppDomain_AssemblyResolve проверку на название необходимой библиотеки. Чем больше библиотек, тем больше условий
5) ПОпытаться исправить ошибки, связанные с различиями версий NET framework путём удаления дублирующихся ссылок (смотря какие ошибки)
